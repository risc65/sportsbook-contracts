# Sportsbook Contracts

API contract definitions for the Sportsbook System.

## Structure

```
├── proto/        # gRPC Protocol Buffer definitions
└── openapi/      # OpenAPI/Swagger specifications
```

## Usage

### Protocol Buffers

Place `.proto` files in the `proto/` directory. Generate code using your preferred protoc plugins.

### OpenAPI

Place OpenAPI specs (`.yaml` or `.json`) in the `openapi/` directory.
